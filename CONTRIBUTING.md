# Contributing to ExcaliburFX

We welcome contributions from anyone! Here’s how you can help:

## Reporting Bugs

- Check if the issue already exists before creating a new one.  
- Include clear reproduction steps and, if possible, screenshots or screen recordings.  

## Suggesting Features

- Open an issue with a detailed description of the feature and why it’s useful.  
- Include any use cases or examples if possible.  

## Pull Requests

- Fork the repository.  
- Create a new branch for your feature or fix.  
- Test your changes thoroughly.  
- Submit a pull request with a clear description of what was changed and why.  

Thank you for helping make ExcaliburFX better!
