# ExcaliburFX

ExcaliburFX est une extension After Effects conçue pour simplifier et accélérer le workflow des créateurs.  
Elle propose des outils puissants pour automatiser certaines tâches, optimiser la gestion des compositions et enrichir les animations, tout en restant intuitive et légère.

---

## Features

- **Curves**
  - Quickly create, save, and apply custom Bézier curves  

- **Presets & Instant**
  - Presets and effects panel to save and apply effects faster  

- **Autoramp**
  - Automatic speed ramp creation  

- **WorkFlow**
  - Quick management of manual tools such as speed, layers, and compositions  
  - Workflow optimization to save time on complex projects  

- **Effects**
  - Fast application and customization of effects on layers  
  - Compatible with native and third-party effects  

- **Presets**
  - Advanced management of user-specific After Effects presets  
  - Instant save and recall of effect configurations  

- **Shakes**
  - Tools to add realistic and controlled camera shakes  
  - Pre-made settings with instant preview  
  - Manual shake creation with live preview  

- **Assets**
  - Organize and manage assets directly from the panel (assets need to be provided by the user)  
  - Live preview of assets  

- **Settings**
  - Customize the interface and keyboard shortcuts  
  - Choose visual themes and user preferences  

- **Script**
  - Built-in scripts to simplify long or complex tasks (apply scripts to layers)  
  - Option to add custom scripts (user-provided)  

- **Misc (Notes & Shortcut)**
  - Write your own notes  
  - Manage and display custom keyboard shortcuts  

- **Searchbar**
  - Search effects and presets directly on the timeline to apply them quickly  

---

## Technologies

- JavaScript / JSX pour After Effects  
- Extendscript  
- Panel HTML/CSS pour interface utilisateur  
- JSON pour gestion des presets et configurations
